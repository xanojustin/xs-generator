# Find the Difference

## Problem
Given two strings `s` and `t` where `t` is generated by randomly shuffling string `s` and then adding one more letter at a random position, find the letter that was added to `t`.

## Structure
- **Run Job (`run.xs`):** Calls the solution function with test inputs
- **Function (`function/find_the_difference.xs`):** Contains the solution logic

## Function Signature
- **Input:**
  - `s` (text, optional): The original string
  - `t` (text): The shuffled string with one extra character added
- **Output:** (text) The character that was added to `t`

## Approach
The solution uses a frequency counting approach:
1. Count occurrences of each character in string `s`
2. Subtract occurrences of each character in string `t`
3. The character with a negative count (or count that goes below zero) is the added character

This approach handles all edge cases including:
- Empty string `s` (the extra character is the only character in `t`)
- The extra character being the same as an existing character
- The extra character being at any position in `t`

## Test Cases
| Input | Expected Output |
|-------|-----------------|
| s: "abcd", t: "abcde" | "e" |
| s: "", t: "a" | "a" |
| s: "a", t: "aa" | "a" |
| s: "ae", t: "aea" | "a" |
| s: "xyz", t: "xzyz" | "z" |

## Alternative Approaches
- **XOR approach:** XOR all characters in both strings - the result is the extra character
- **Sum approach:** Sum ASCII values of characters in both strings and find the difference
